# 2D World RPG

A simple 2D RPG game built with HTML5 Canvas and JavaScript.

## Description

This is a 2D world RPG game featuring a hero character that can move around in a tile-based world. The game includes background and foreground layers, and supports basic input controls for movement.

## Features

- Tile-based world rendering
- Hero character with sprite animation
- Background and foreground layers
- Keyboard input handling
- Canvas-based graphics

## How to Run

1. Clone or download the repository
2. Open `index.html` in a modern web browser
3. The game will start automatically

## Technologies Used

- HTML5 Canvas
- JavaScript (ES6 modules)
- CSS

## Project Structure

- `index.html` - Main HTML file
- `main.js` - Main game logic
- `images/` - Game assets (background, foreground, hero sprites)
- `package.json` - Project metadata
