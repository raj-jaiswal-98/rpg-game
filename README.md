# 2D World RPG

A simple 2D RPG game built with HTML5 Canvas and TypeScript.

## Description

This is a 2D world RPG game featuring a hero character that can move around in a tile-based world. The game includes background and foreground layers, and supports basic input controls for movement.

## Features

- Tile-based world rendering
- Hero character with sprite animation
- Background and foreground layers
- Keyboard input handling
- Canvas-based graphics

## How to Run

### Development Mode

1. Clone or download the repository
2. Install dependencies: `npm install`
3. Start the development server: `npm run dev`
4. Open your browser to `http://localhost:3000`
5. The game will start automatically with live reload

### Production Build

1. Build the project: `npm run build`
2. Open `index.html` in a modern web browser
3. The game will start automatically

## Technologies Used

- HTML5 Canvas
- TypeScript
- JavaScript (ES6 modules)
- CSS

## Project Structure

- `index.html` - Main HTML file
- `main.ts` - Main game logic (TypeScript)
- `.scripts/` - Game classes (TypeScript)
- `images/` - Game assets (background, foreground, hero sprites)
- `dist/` - Compiled JavaScript files
- `package.json` - Project metadata
